# Family Pay

## Build

### Pre-requisites
```
npm i -g ionic cordova native-run ios-deploy cordova-res
ionic cordova platform rm ios
ionic cordova platform add ios
```

### Build locally

```
ionic serve
```

### Build for mobile
```
ionic cordova run ios
```
